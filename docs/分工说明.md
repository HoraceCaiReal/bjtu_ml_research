## 任务衔接说明（初稿，项目完成后按实际填写）

1. **第一阶段（第1-2周）**：仓库管理员搭建 GitHub 仓库与 Anaconda 环境，协作者克隆后配置环境，全组确认环境一致。
2. **第二阶段（第3-4周）**：数据预处理方法实现（CLAHE + 高斯 + 中值滤波）；特征提取（HOG + LBP + GLCM + 边缘密度）；传统机器学习三分类器（决策树/SVM/KNN）实验；无监督聚类（K-Means/GMM/DBSCAN 核心 + Agglomerative/谱聚类补充）实验。
3. **第三阶段（第5周）**：深度学习 CNN 模型训练与调参；各组员推送代码到 `main` 分支；在 `notebooks/05_comparison.ipynb` 中进行所有方法的结果对比分析。
4. **第四阶段（第6周）**：全组共同整合 Jupyter 报告（5 个 notebook），录制操作视频，检查提交物。
